Cursos realizados

Desarrollador Java EE Front-End Multiplataforma

Curso realizado en el Centro de Formación en Tecnologías de la Información y las Comunicaciones con una duración de 265 horas. En este curso, el Back-End de las aplicaciones que he desarrollado ha sido con Java, JavaEE, Spring, Spring Boot y Microservicios mientras que el Front-End he utilizado JavaScript, Angular y AngulasJS.

El patrón de arquitectura que he utilizado ha sido el Modelo-Vista-Controlador (MVC) donde la lógica de negocio se encuentra en los 'services', la comunicación con la base de datos se realiza con los 'repositories o DAO' y por último tenemos un controlador que se comunica con las vistas.

Asimismo, durante este curso he trabajado con Hibernate (ORM) framwork que permite mapear las tablas de la base de datos en Entidades. También he realizado prácticas con Spring Data JPA (Java Persistence API) que facilita un módulo para el acceso a las bases de datos. Las bases de datos con las que he trabajado han sido MySQL, SQL y MongoDB. Para las dos primeras he implementado la interfaz de JpaRepository y para MongoDB existe una similar, MongoRepository.

En la capa de la Vista he trabajdo con JSP (Java Server Pages) que permite incrustar código Java detro del html. También he realizado algunas páginas con la ayuda de JSTL (Java Standard Tag Library). Actualmente hay un nuevo motor de plantillas Java del lado servidor para entornos web que se conoce como Thymeleaf.

He podido conocer y realizar numerosas prácticas utilizando la arquitectura de microservicios, es decir, construir una aplicación como un conjunto de pequeños servicios que funcionan de forma autónoma y comunicandose a través del protocolo HTTP. Para realizar la comunicación entre los microservicios he utilizado la clase RestTemplate. Esto lo he realizado con Spring clásico con su configuración de XML o clase Java de configuración y también lo he realizado microservicios con Spring Boot. Finalizando el curso puede realizar una breve introducción con Docker, donde pudimos crear imagenes de nuestras aplicaciones y luego instanciarla en forma de contenedor.
